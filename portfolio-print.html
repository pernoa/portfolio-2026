왜 테크타카인가
01

B2B SaaS 퍼널 → 아르고 MQL→계약에 즉시 적용

올거나이즈에서 GTM·Mixpanel 기반 전환 퍼널을 0에서 설계·구축. 아르고의 MQL→SQL→기회→계약 퍼널에 바로 기여 가능.

02

0→1 마케팅 조직 빌딩 → 신설 조직에 최적

두 차례 팀 처음부터 구성·리드. OKR·PDCA 도입, 팀 성과관리 체계를 0에서 만든 경험.

03

복잡한 도메인 빠른 적응 → 물류 SaaS도 같은 방식

AI·글로벌 SaaS·공공미술 등 새 도메인에 빠르게 적응. 데이터 기반 접근이 도메인을 타지 않음.
